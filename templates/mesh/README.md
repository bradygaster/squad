# Mesh Sync Templates

Scaffolding files for distributed squad coordination. Copy these to your project root and customize `mesh.json.example` with your squad configuration. Run `sync-mesh.sh` (bash) or `sync-mesh.ps1` (PowerShell) before agents read remote state.

See `.squad/skills/distributed-mesh/SKILL.md` for full documentation on zones, write partitioning, and trust boundaries.
